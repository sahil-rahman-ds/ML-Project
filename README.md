## Student Math Score Prediction - End-to-End ML Project

To predict student math scores based on various features. It features an end-to-end pipeline complete with modular components for data ingestion, data transformation, and model training. 
## Features

* **Modular Architecture:** Clean and maintainable codebase separated into reusable components (Ingestion, Transformation, Training).
* **Custom Exception & Logging:** Robust error handling and tracking using custom logger and exception modules.
* **Web Interface:** User-friendly frontend built with HTML templates and served via a Python web framework (e.g., Flask) in `application.py`.
* **Automated Pipelines:** Dedicated prediction and training pipelines.
* **Cloud Ready:** Includes `.ebextensions` configuration for seamless deployment to AWS Elastic Beanstalk.

Installation and Setup
1. Clone the repository:

Bash
git clone [https://github.com/sahil-rahman-ds/ML-Project]
cd your-repo-name
2. Create a virtual environment:

Bash
python -m venv venv
3. Activate the virtual environment:

Windows:

Bash
venv\Scripts\activate
Mac/Linux:

Bash
source venv/bin/activate
4. Install dependencies:

Bash
pip install -r requirements.txt
5. (Optional) Install the project as a local package:

Bash
python setup.py install
Usage
To run the web application locally:

Ensure your virtual environment is activated.

Execute the main application file:

Bash
python application.py
Open your web browser and navigate to http://127.0.0.1:5000 (or the port specified in your console).

To trigger the training pipeline manually:
You can run the training pipeline directly to generate new models and preprocessor objects in the artifacts/ folder:

Bash
python src/pipeline/train_pipeline.py
Logging and Debugging
All application logs are saved in the logs/ directory. If you encounter any issues, check the latest log file for detailed tracebacks generated by the custom exception.py handler.
